# FinFlow — Sistema de Finanças Pessoais 💰

## Como usar

1. **Abra o arquivo `index.html`** diretamente no seu navegador (Chrome, Firefox, Edge, Safari).
   - Não precisa de servidor, instalação ou internet (após carregar as fontes).

---

## Funcionalidades

### 📊 Painel
- Resumo do mês: receitas, despesas, saldo e contas pendentes
- Comparação com mês anterior (variação %)
- Gráfico de evolução mensal do ano
- Gastos por categoria com barras de progresso
- Últimos lançamentos do mês

### ➕ Lançamentos
- Adicione receitas e despesas
- Filtro por mês, tipo e categoria
- Edite ou exclua qualquer lançamento
- Observações opcionais

### 📋 Contas a Pagar
- Cadastre contas com data de vencimento
- Recorrência: única, mensal ou anual
- Marque como paga com um clique
- Alertas visuais para contas vencidas
- Filtro: todas / pendentes / pagas

### 📈 Visão Anual
- Totais de receita, despesa e saldo do ano
- Gráfico de barras + linha de saldo
- Tabelas mensais de receitas e despesas
- Selecione qualquer ano

### 📉 Relatório
- Gráfico de rosca com distribuição de despesas
- Legenda com percentuais por categoria
- Ranking das maiores despesas do ano

### 🏷️ Categorias
- Crie categorias personalizadas
- Escolha cor e ícone emoji
- 8 categorias padrão incluídas

### ⚙️ Configurações
- Exportar dados (backup JSON)
- Importar dados (restaurar backup)
- Limpar todos os dados
- Alternar tema claro/escuro

---

## Dados

Todos os dados são **salvos automaticamente** no `localStorage` do navegador.  
Use **Exportar JSON** para fazer backup e **Importar JSON** para restaurar.
